Objetivo de este trabajo

El propósito de este trabajo de visualización es el de mostrar la estadística recolectada de las fotos / videos de mi sobrino, Benjamín, en el lapso de 365 días. Ese lapso inicia desde el 11/07/2020, cuando nació, hasta el 11/07/2021, cuando cumplió su primer año de edad. Las estadísticas fueron recolectadas directamente desde el File System de mi computadora personal. La herramienta utilizada para tal fin fue "File List". A continuación, dejo el link a quien le interese descargarla y utilizarla .
